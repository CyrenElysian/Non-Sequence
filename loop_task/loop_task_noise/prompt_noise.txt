Your task is to augment the existing data by expanding each state description into a short, multi-sentence scene snapshot. This snapshot will be used to test a model's ability to identify the key signal among natural contextual noise.

## Data Template
{
  "id": number,
  "sceniro": "scenario description",
  "steps": ["step 1", "step 2", ...],
  "loop_idx": index of the loop step (note: indexing may start from 0),
  "loop_step": "content of the loop step",
  "descriptions": [
    ["state description text", answer (0/1/2), difficulty ("easy"/"medium"/"hard"/"na")],
    ...
  ]
}

## Field Explanation
- Answer: 0 = exit loop, 1 = continue loop, 2 = irrelevant to loop decision
- Difficulty: easy, medium, hard, na (na is used only when answer = 2)

## Core Requirement
For each description entry, rewrite the first field (the state description text) into a short paragraph of 4–6 sentences. The paragraph must:
1. Lead to exactly the same loop decision (0/1/2) as the original.
2. Keep the original difficulty level (do not alter the difficulty field).
3. Contain **exactly one sentence** that carries the original core signal — the fact that determines whether the loop should stop, continue, or is irrelevant.
4. The other 3–5 sentences must be **noise**: natural, scene-appropriate details that have **no causal bearing** on the loop's termination condition.

## What is NOISE in this multi-sentence setting
Noise sentences should come from **dimensions unrelated to the loop condition**. Choose from the following dimensions, specific to the current scenario:
- **Environment**: lighting, temperature, weather, background sounds, smells, textures (e.g., "The afternoon sun casts long shadows across the floor.")
- **Physical state / posture**: what the person is doing with their body that is irrelevant to the loop (e.g., "She shifts her weight from one foot to the other.")
- **Concurrent unrelated events**: minor things happening nearby that do not affect the loop (e.g., "A bus rumbles past on the street outside.")
- **Object properties**: non-essential attributes of items in the scene (e.g., "The mug on the table has a small chip on its rim.")
- **Mental state / idle thoughts** (only if they do not relate to the loop): e.g., "He wonders what to have for dinner later."

**Noise sentences must NOT:**
- Provide any additional evidence for or against the loop condition.
- Explain, confirm, or quantify the core signal.
- Contain dynamic events that strengthen a static observation.
- Be completely generic (e.g., "The sun is shining") unless tied to the specific scene.

## The Core Signal Sentence
- This sentence must be clearly recognizable as a statement of fact that, **on its own**, is sufficient to decide the loop's state.
- It should be the **unchanged original state description** or a very close paraphrase that preserves its exact reasoning strength. Do **not** add emphasis, explanations, or adjectives that make the signal stronger or weaker.
- You may adjust phrasing slightly to fit grammatically into the paragraph (e.g., change "the water is not boiling" to "The water in the pot is still not boiling"), but the logical content must remain identical.
- The core sentence can appear at any position in the paragraph (beginning, middle, or end). Vary its position across entries to prevent predictability.

## Special Handling for answer=2 (na)
- The original na description is already irrelevant to the loop decision. In the expanded paragraph, **all sentences** must remain irrelevant.
- Build a coherent scene around the original na detail, but do not introduce any information that could accidentally imply a loop state.
- Example: Original: "the floor tiles are a light beige color" → Paragraph: "The floor tiles are a light beige color, with a subtle geometric pattern. A faint hum comes from the air conditioning. The reception desk is made of dark wood. A magazine lies open on a side table."

## Deletion Test
After writing the paragraph, mentally remove all noise sentences. The remaining core signal sentence should be exactly as strong and clear as the original description. If the paragraph makes the signal easier to judge, the noise is incorrectly written.
